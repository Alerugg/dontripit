Qué es Dontripit

Una capa de datos y descubrimiento para el ecosistema TCG moderno.

No estamos creando solo un catálogo. Estamos construyendo la capa de datos y descubrimiento sobre la que podrá crecer una experiencia completa para coleccionistas, tiendas y herramientas de terceros.

Producto en construcción, dirección clara

Dontripit nace con enfoque infra/producto: calidad de estructura, legibilidad de datos y una UX capaz de escalar hacia funcionalidades reales sin rehacer la plataforma en cada etapa.

Visión de plataforma multi-juego

El objetivo es consolidar una base transversal para distintos TCG, permitiendo crecimiento ordenado hacia colección, wishlist y, finalmente, marketplace especializado.

La visión a largo plazo

Primero infraestructura fiable. Después, una experiencia de mercado de nueva generación.

La V1 se centra en catálogo, búsqueda y API. La visión final es más ambiciosa: evolucionar hacia un marketplace TCG moderno, claro y multi-juego, con una base de datos realmente robusta detrás.

Queremos combinar profundidad técnica y usabilidad real para crear una alternativa más limpia, especializada y escalable para coleccionistas, tiendas y productos de terceros.
